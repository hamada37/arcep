DEMANDES

INSTRUCTIONS


Vous pouvez effectuer vos diverses demandes via le formulaire ci-dessous.

  1. Téléchargez le document qui correspond à votre besoin.
  2. Réunissez les éléments demandés et complétez le formulaire papier téléchargé
  3. Saisissez le formulaire en ligne
  4. Joignez les documents obligatoires dans la section « Fichiers ».
  5. Si vos documents ont un poids jusqu’à  2 Mégas, cliquer sur le bouton « Parcourir »
  6. Si vos documents dépassent 2 Mégas, il est préférable d’utiliser un site de transfert de fichier : https://www.wetransfer.com/
  1. Collez le lien fourni par wetransfer dans le champ « Liens des fichiers »
